# 🧠 Landing Page - Neuropsicopedagogia

Projeto de landing page desenvolvido com foco em atendimento profissional na área de neuropsicopedagogia.

## 🚀 Objetivo

Criar um site moderno, responsivo e com foco em conversão, facilitando o contato direto via WhatsApp.

## 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📱 Funcionalidades

- Layout responsivo (mobile e desktop)
- Botão de contato direto via WhatsApp
- Seções organizadas (Sobre, Serviços, Depoimentos e Contato)
- Animações suaves ao rolar a página
- Design focado na experiência do usuário

## 🎯 Destaques do projeto

- Estrutura de landing page profissional
- Design moderno e limpo
- Uso de boas práticas de UI/UX
- Foco em conversão (call-to-action estratégico)
